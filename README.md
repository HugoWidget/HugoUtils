# HugoUtils

## 项目介绍

Hugo系列核心库

## 功能

1. 希沃管家关闭与重启
2. 希沃管家屏幕锁管理
3. 希沃冰点服务交互
4. 希沃冰点管理
5. 希沃文件保护服务管理
6. 希沃虚拟磁盘挂载
7. 希沃映像劫持
8. 希沃下载与卸载
9. 希沃管家密码找回
10. 禁用希沃更新

## 环境依赖

Visual Studio 2022

## 项目依赖

[WinUtils](https://github.com/howdy213/WinUtils)(Windows微功能组件库，已内置)

[cpp-httplib](https://github.com/yhirose/cpp-httplib)(HTTP/HTTPS网络库，依赖OpenSSL，已内置)

[hash-library](https://github.com/stbrumme/hash-library)(哈希库，已内置)
